# floppercon

[![Travis CI](https://travis-ci.org/jessfraz/floppercon.svg?branch=master)](https://travis-ci.org/jessfraz/floppercon)

 Get tweets for a hashtag
 Version: v0.1.0

  -v    print version and exit (shorthand)

        print version and exit
