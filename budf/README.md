# budf

Backup dotfiles to s3, and show diff.

```console
$ budf -h
Usage of budf:
  -bucket string
        AWS S3 bucket, overrides value in .budfrc
  -d    run in debug mode
  -key string
        AWS API Key, overrides value in .budfrc
  -region string
        AWS S3 bucket region, overrides value in .budfrc
  -reset
        reset the config file values
  -secret string
        AWS Secret, overrides value in .budfrc
  -v    print version and exit (shorthand)
  -version
        print version and exit
```
